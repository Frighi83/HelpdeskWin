# HelpdeskWin  
Gestione ticket semplice, veloce e completamente locale — senza cloud, senza abbonamenti.

HelpdeskWin è un’applicazione Windows progettata per trasformare email dei clienti in ticket gestibili, con un’interfaccia pulita e un flusso di lavoro immediato.  
Perfetta per tecnici, negozi informatici, freelance, piccole aziende e chiunque gestisca assistenza tramite email.

---

## 🚀 Caratteristiche principali

- 📩 **Importazione automatica email** (IMAP o SMTP)
- 🎫 **Creazione ticket immediata**
- 🗂️ **Archivio ticket locale**
- 🔐 **Nessun cloud, nessun server esterno**
- 💾 **Supporto SQLite o SQL Server**
- 🖥️ **Applicazione Windows leggera e veloce**
- 🆓 **Completamente gratuita**
- 🛒 **Disponibile sul Microsoft Store**

## ⚙️ Configurazione

### IMAP
- Email  
- Password  
- Server IMAP  
- Porta  

### SMTP
- Server SMTP  
- Porta  
- Username  
- Password  
- SSL opzionale  

### Database
- SQLite (default, zero configurazione)  
- SQL Server (per ambienti professionali)

Nessun invio verso server esterni.  
Nessun tracciamento.  
Nessun cloud.
